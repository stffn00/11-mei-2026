About The Website

Website adalah sekumpulan halaman digital yang saling terhubung dan dapat diakses melalui internet.

HTML

Berfungsi untuk membuat struktur dan konten dasar halaman bisa juga sebagai struktur utama website.

CSS

digunakan untuk mendesain tampilan visual dan layout dan juga mengatur tampilan agar menarik dan responsif.

JavaScript

menambahkan interaktivitas dan fungsi dinamis yang membuat halaman web lebih responsif dan menarik bagi pengguna dan mengatur perpindahan halaman tanpa reload.
